# 截图目录

将 README 用到的界面截图放在本目录，建议命名：

| 文件名 | 建议内容 |
|--------|----------|
| `login.png` | 登录页 |
| `workbench.png` | 工作台 |
| `project-detail.png` | 项目详情（任务/甘特等） |
| `admin.png` | 管理后台（可选） |

格式：PNG 或 JPG；宽度约 1200～1600px 即可，单张尽量 &lt; 500KB 便于克隆仓库。

在根目录 `README.md` 中引用示例：

```markdown
![登录页](docs/images/login.png)
```
